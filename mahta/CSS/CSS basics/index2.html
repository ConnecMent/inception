Basics of HTML

This is one of the simplest examples of HTML coding:



!DOCTYPE html

Defines the document type and version of HTML being used (HTML5 in this case).

html

This specifies the part of our entire code that belongs to html coding.

head

Contains metadata about the HTML document, such as the title and linked stylesheets.

title

Specifies the title of the HTML document, which appears in the browser's title bar or tab.

body

Represents the main content of the HTML document.

p & h1

p defines a paragraph, a block of text, while h1 to h6 represent the highest to lowest level heading, commonly used for page titles or major section headings.(h1's font size is bigger than h2 and ...)
